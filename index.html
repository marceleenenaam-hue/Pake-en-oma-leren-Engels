Als je de foto liever in een **images** map wilt zetten: 1. Maak eerst een map `images` in je repository 2. Upload de foto daar 3. Gebruik dan deze URL: ``` https://raw.githubusercontent.com/marceleenenaam-hue/Pake-en-oma-leren-Engels/main/images/pake-inge-vakantie.jpg ``` --- ## Volledige body CSS met overlay (voor betere leesbaarheid) ```css body { margin: 0; font-family: 'Poppins', sans-serif; background-image: url('https://raw.githubusercontent.com/marceleenenaam-hue/Pake-en-oma-leren-Engels/main/pake-inge-vakantie.jpg'); background-size: cover; background-position: center; background-attachment: fixed; background-repeat: no-repeat; min-height: 100vh; display: flex; justify-content: center; align-items: center; padding: 20px 10px; position: relative; } /* Lichte overlay zodat de tekst goed leesbaar blijft */ body::before { content: ''; position: fixed; top: 0; left: 0; right: 0; bottom: 0; background: rgba(135, 206, 235, 0.3); /* Lichtblauwe tint */ backdrop-filter: blur(2px); /* Lichte blur voor zachter effect */ z-index: -1; } ``` --- ## Controleer of het werkt Na het uploaden kun je testen of de URL werkt door deze in je browser te plakken: ``` https://raw.githubusercontent.com/marceleenenaam-hue/Pake-en-oma-leren-Engels/main/pake-inge-vakantie.jpg ``` Als je de foto ziet, werkt de URL! ðŸ“¸ --- Wil je dat ik de **complete HTML code** geef met de foto al ingebouwd? ðŸŒ´
